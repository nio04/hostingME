               //html part
               

              // css part
                .responsive-video-wrapper {
                  position: relative;
                  padding-bottom: 56.25%; /* Aspect Ratio: 16:9 */
                  height: 0;
                  overflow: hidden;
                }

                .responsive-video-wrapper iframe,
                .responsive-video-wrapper object,
                .responsive-video-wrapper embed,
                .responsive-video-wrapper video {
                  position: absolute;
                  top: 0;
                  left: 0;
                  width: 100%;
                  height: 100%;
                }
  • responsive-video-wrapper ` creates a containing block with a `padding-bottom` of `56.25%`, representing the aspect ratio of 16:9    , it maintains the aspect ratio as the screen size changes.
  • video element is then absolutely positioned inside this container, taking up the full width and height, ensuring it scales responsively.
