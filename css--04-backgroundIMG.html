background image

single background image

              
               .container {
                background-image: url('background.jpg');
              }

multiple background image

              
                .container {
                  background-image: url('background1.jpg'), url('background2.jpg');
                }

gradient background image

              
               .container {
                  background-image: linear-gradient(to bottom, #ff0000, #00ff00);
               }

svg background image

              
               .container {
                background-image: url('image.svg');
              }

image from external source

              
              .container {
                background-image: url('https://example.com/background.jpg');
              }

linear gradient and background image mix

              
              .container {
               background-image: linear-gradient(to right, rgba(0, 0, 255, 0.5), rgba(255, 255, 0, 0.5)),
                  url("../../media/examples/lizard.png");
              }

additional property for background image

background-repeat

  • repeat
  • repeat-x
  • repeat-y
  • no-repeat
              
              .container {
                background-image: url('background.jpg');
                background-attachment: fixed;
              }

background-size

  • auto
  • cover: image will cover the entire container without stretching, possibly cropping parts of the image
  • contain: image will be contained within the container, scaling as needed without cropping 
              
             .container {
              background-image: url('background.jpg');
              background-size: cover;
            }

background-position

  • top
  • bottom
  • left
  • right
  • center
              
             .container {
              background-image: url('background.jpg');
              background-position: center;
            }

background-attachment

  • scroll: background image scrolls along with the content
  • fixed: background image remains fixed within the viewport as the content scrolls 
              
                .container {
                  background-image: url('background.jpg');
                  background-attachment: fixed;
                }

background-blend-mode

This property specifies how the background image should blend with the background color or other background images

  • normal
  • multiply
  • screen
  • overlay
  • darken
  • lighten
  • color-dodge
  • color-burn
  • difference
  • exclusion
              
               .container {
                  background-image: url('background.jpg');
                  background-color: rgba(0, 0, 0, 0.5);
                  background-blend-mode: multiply;
                }

background-origin

This property specifies the positioning area of the background images

  • border-box
  • padding-box
  • content-box
              
               .container {
                  background-image: url('background.jpg');
                  background-origin: padding-box;
                }
