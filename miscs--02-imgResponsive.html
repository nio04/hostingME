html

srcset attribute

to instruct the browser to use different images at different resolutions.
srcset: defines a list of different images and their widths as w

picture element
  • The browser first checks the conditions within the source elements from top to bottom.
  • If the conditions are met (e.g., viewport width is at least 800px), it loads that source's `srcset`
  • If no conditions are met, the img element's `src` is used.

css

keep aspect ratio

              
              img {
                max-width: 100%;
                height: auto;
              }
  • max-width: 100% ensures that the image can scale down if it needs to, but never scales up to be larger than its original size.
  • height: auto maintains the aspect ratio of the image as it scales down

media query

              
                .background-image {
                  background-image: url('small.jpg');
                  background-size: cover;
                }

                @media (min-width: 800px) {
                  .background-image {
                    background-image: url('large.jpg');
                  }
                }
  • by default, use small image
  • when the viewport is greater than 800px, use large image
