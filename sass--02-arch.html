architecture template

              
                ```
                sass/
                |
                |– base/
                |   |– _reset.scss       # Reset/normalize
                |   |– _typography.scss  # Typography rules
                |   |- index.scss        #
                |   ...                  # Etc.
                |
                |– components/
                |   |– _buttons.scss     # Buttons
                |   |– _carousel.scss    # Carousel

                |   |- _list-design.scss # list design
                |   |- _link-design.scss # link design
                |   |- _cards.scss       # cards
                |   |- index.scss        # index file
                |   ...                  # Etc.
                |
                |– layout/
                |   |– _grid.scss        # Grid system
                |   |– _header.scss      # Header section
                |   |– _footer.scss      # Footer section
                |   |– _sidebar.scss     # Sidebar section
                |   |– _forms.scss       # Form styles
                |   |- index.scss        # index file
                |   ...                  # Etc.
                |
                |– pages/
                |   |– _home.scss        # Home specific styles
                |   |– _about.scss       # About specific styles
                |   |- index.scss        # index file
                |   ...                  # Etc.
                |
                |– themes/
                |   |– _theme.scss       # Default theme
                |   |– _admin.scss       # Admin theme
                |   |- _night-theme.scss #night theme
                |   |- index.scss        # index file
                |   ...                  # Etc.
                |

                |   |– _variables.scss   # Sass Variables
                |   |– _functions.scss   # Sass Functions
                |   |– _mixins.scss      # Sass Mixins
                |   |– _placeholders.scss# Sass Placeholders
                |   |- _anim.scss        # animations
                |   |- _bp.scss          # breakpoints
                |   |- index.scss        # index file

                |– vendors/
                |   |– _bootstrap.scss   # Bootstrap
                |   |– _jquery-ui.scss   # jQuery UI
                |   |- index.scss        # index file
                |   ...                  # Etc.
                |
                ```

architecture explain

architecture explain

  • base: Contains global styles, such as resets and typographical rules, which are applied throughout site
  • components: Holds all the styles for buttons, carousels, sliders, etc., making them modular and reusable
  • layout: Contains styling for the main parts of the site or application such as the header, footer, or grid system.
  • pages: Page-specific styling
  • theme: for different theme
  • abstracts: helper tools and settings, like variables, mixins, and functions.
  • vendor: Includes all the CSS styles from external libraries and frameworks - normalize, Bootstrap, jQuery UI, etc.
